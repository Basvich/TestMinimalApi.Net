# Aspire9Test.AppHost

Este proyecto es el orquestador principal de la solución Aspire. Se encarga de definir y lanzar los distintos servicios (backend, frontend, etc.) de la aplicación, gestionando la configuración de recursos, dependencias y el ciclo de vida de la solución durante el desarrollo y pruebas. No contiene lógica de negocio ni interfaz de usuario.
